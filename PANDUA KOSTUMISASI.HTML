Gambaran Umum

Aplikasi Brangkas File Data Guru adalah sistem manajemen data guru yang komprehensif dengan fitur-fitur canggih untuk mengelola data personal, berkas, dan keamanan sistem.

Fitur Utama

  • Multi-user authentication
  • File upload & management
  • Advanced search & filter
  • Backup & restore system
  • Admin dashboard
  • Login history tracking

Teknologi

  • HTML5 + CSS3
  • JavaScript (ES6+)
  • Tailwind CSS
  • Font Awesome Icons
  • LocalStorage API
  • Responsive Design

Persyaratan Sistem

Browser: Chrome 80+, Firefox 75+, Safari 13+, Edge 80+

JavaScript: Harus diaktifkan

LocalStorage: Minimal 10MB tersedia

Resolusi: Minimal 1024x768 (optimal: 1920x1080)

Kostumisasi Dasar

1. Mengubah Skema Warna

Aplikasi menggunakan sistem warna Tailwind CSS yang mudah dikustomisasi. Berikut panduan lengkap untuk mengubah skema warna:

📋 Langkah-langkah Detail:

  1. 1. Buka file HTML aplikasi dengan text editor (VS Code, Notepad++, dll)
  2. 2. Gunakan fitur Find & Replace (Ctrl+H) untuk mengganti warna secara massal
  3. 3. Ganti semua instance warna biru dengan warna pilihan Anda
  4. 4. Simpan file dan refresh browser untuk melihat perubahan
/* PANDUAN LENGKAP PENGGANTIAN WARNA */

/* 1. WARNA UTAMA - Tombol dan Elemen Aktif */
bg-blue-600 → bg-purple-600    /* Background tombol utama */
bg-blue-700 → bg-purple-700    /* Background tombol saat hover */
bg-blue-500 → bg-purple-500    /* Background elemen sekunder */
bg-blue-800 → bg-purple-800    /* Background elemen gelap */

/* 2. WARNA TEKS */
text-blue-600 → text-purple-600 /* Teks link dan judul */
text-blue-700 → text-purple-700 /* Teks tebal */
text-blue-800 → text-purple-800 /* Teks heading */

/* 3. WARNA BORDER */
border-blue-600 → border-purple-600 /* Border utama */
border-blue-500 → border-purple-500 /* Border input focus */
border-blue-300 → border-purple-300 /* Border ringan */

/* 4. WARNA HOVER EFFECTS */
hover:bg-blue-700 → hover:bg-purple-700
hover:text-blue-600 → hover:text-purple-600
hover:border-blue-600 → hover:border-purple-600

/* 5. WARNA FOCUS STATES */
focus:ring-blue-500 → focus:ring-purple-500
focus:border-blue-500 → focus:border-purple-500

/* 6. GRADIENT BACKGROUNDS */
from-blue-50 → from-purple-50
to-indigo-100 → to-purple-100

/* CONTOH IMPLEMENTASI CUSTOM CSS */
.btn-primary {
    @apply bg-purple-600 hover:bg-purple-700 text-white px-4 py-2 rounded-lg transition-colors;
}

.input-field {
    @apply border border-gray-300 focus:border-purple-500 focus:ring-2 focus:ring-purple-200;
}

/* CUSTOM GRADIENT UNTUK HEADER */
.header-gradient {
    background: linear-gradient(135deg, #8b5cf6 0%, #a855f7 50%, #c084fc 100%);
}

/* CUSTOM SHADOW DENGAN WARNA TEMA */
.shadow-theme {
    box-shadow: 0 4px 14px 0 rgba(139, 92, 246, 0.15);
}

Pilihan Warna Populer

  • Hijau: green-600, emerald-600
  • Merah: red-600, rose-600
  • Ungu: purple-600, violet-600
  • Orange: orange-600, amber-600
  • Teal: teal-600, cyan-600

Tips Penting

  • Gunakan warna yang kontras untuk aksesibilitas
  • Test pada berbagai perangkat
  • Simpan backup sebelum mengubah
  • Konsisten dalam penggunaan shade (600, 700, dll)

Contoh Penerapan Tema Sekolah

Tema Hijau (Alam/Lingkungan):

Ganti semua blue-600 dengan green-600

Tema Merah (Nasionalisme):

Ganti semua blue-600 dengan red-600

Hasil: Aplikasi akan memiliki identitas visual yang sesuai dengan tema sekolah Anda.

Referensi: Kunjungi Tailwind Color Palette untuk melihat semua pilihan warna dan shade yang tersedia.

document.getElementById('teacherForm').addEventListener('submit', function(e) {
    e.preventDefault();
    
    const formData = new FormData(this);
    
    // Data guru dengan field baru
    const teacherData = {
        id: Date.now(),
        // Field existing
        namaGuru: formData.get('namaGuru'),
        nik: formData.get('nik'),
        nip: formData.get('nip'),
        // ... field lainnya ...
        
        // FIELD BARU - Tambahkan di sini
        nomorTelepon: formData.get('nomorTelepon'),
        emailPribadi: formData.get('emailPribadi'),
        asalDaerah: formData.get('asalDaerah'),
        catatanKhusus: formData.get('catatanKhusus'),
        jamMengajar: parseInt(formData.get('jamMengajar')) || 0,
        tanggalMulaiMengajar: formData.get('tanggalMulaiMengajar'),
        
        // Untuk checkbox (multiple values)
        sertifikasi: formData.getAll('sertifikasi[]'),
        
        createdAt: new Date().toISOString(),
        updatedAt: new Date().toISOString()
    };
    
    // Simpan data
    dataManager.saveTeacher(teacherData);
    showNotification('Data guru berhasil disimpan!', 'success');
});

                    <label class="text-sm font-medium text-gray-500">Asal Daerah</label>

                        : '<span class="text-gray-500">Tidak ada sertifikasi</span>'

// 3. UPDATE FUNCTION EDIT TEACHER (untuk mengisi form saat edit)
function editTeacher(teacherId) {
    const teacher = dataManager.getTeacher(teacherId);
    if (!teacher) return;
    
    // Fill existing fields
    document.getElementById('namaGuru').value = teacher.namaGuru || '';
    document.getElementById('nik').value = teacher.nik || '';
    
    // FILL FIELD BARU
    document.getElementById('nomorTelepon').value = teacher.nomorTelepon || '';
    document.getElementById('emailPribadi').value = teacher.emailPribadi || '';
    document.getElementById('asalDaerah').value = teacher.asalDaerah || '';
    document.getElementById('catatanKhusus').value = teacher.catatanKhusus || '';
    document.getElementById('jamMengajar').value = teacher.jamMengajar || '';
    document.getElementById('tanggalMulaiMengajar').value = teacher.tanggalMulaiMengajar || '';
    
    // Fill checkboxes
    if (teacher.sertifikasi && teacher.sertifikasi.length > 0) {
        teacher.sertifikasi.forEach(cert => {
            const checkbox = document.querySelector(`input[name="sertifikasi[]"][value="${cert}"]`);
            if (checkbox) checkbox.checked = true;
        });
    }
}

// 1. UPDATE FUNCTION handleFileUpload untuk berkas baru
function handleFileUpload(input, fileType) {
    const files = Array.from(input.files);
    const statusDiv = document.getElementById(`${fileType}-status`);
    const filesDiv = document.getElementById(`${fileType}-files`);
    
    // Validasi berdasarkan jenis berkas
    const fileValidation = {
        'suratSehat': {
            maxSize: 5 * 1024 * 1024, // 5MB
            allowedTypes: ['application/pdf', 'image/jpeg', 'image/jpg', 'image/png'],
            maxFiles: 1
        },
        'kartuKeluarga': {
            maxSize: 5 * 1024 * 1024, // 5MB
            allowedTypes: ['application/pdf', 'image/jpeg', 'image/jpg', 'image/png'],
            maxFiles: 1
        },
        'sertifikatProfesi': {
            maxSize: 10 * 1024 * 1024, // 10MB
            allowedTypes: ['application/pdf'],
            maxFiles: 5 // Multiple files allowed
        },
        'rekeningBank': {
            maxSize: 5 * 1024 * 1024, // 5MB
            allowedTypes: ['application/pdf', 'image/jpeg', 'image/jpg', 'image/png'],
            maxFiles: 1
        },
        'npwp': {
            maxSize: 5 * 1024 * 1024, // 5MB
            allowedTypes: ['application/pdf', 'image/jpeg', 'image/jpg', 'image/png'],
            maxFiles: 1
        }
    };
    
    const validation = fileValidation[fileType];
    if (!validation) {
        statusDiv.innerHTML = '<span class="text-red-600">Jenis berkas tidak dikenali</span>';
        return;
    }
    
    // Validasi jumlah file
    if (files.length > validation.maxFiles) {
        statusDiv.innerHTML = `<span class="text-red-600">Maksimal ${validation.maxFiles} file</span>`;
        return;
    }
    
    // Validasi setiap file
    for (let file of files) {
        // Validasi ukuran
        if (file.size > validation.maxSize) {
            statusDiv.innerHTML = `<span class="text-red-600">File ${file.name} terlalu besar (max: ${(validation.maxSize / 1024 / 1024).toFixed(1)}MB)</span>`;
            return;
        }
        
        // Validasi tipe file
        if (!validation.allowedTypes.includes(file.type)) {
            statusDiv.innerHTML = `<span class="text-red-600">Format file ${file.name} tidak didukung</span>`;
            return;
        }
    }
    
    // Proses upload
    statusDiv.innerHTML = '<span class="text-blue-600"><i class="fas fa-spinner fa-spin mr-1"></i>Mengupload...</span>';
    
    // Simulasi upload (dalam aplikasi nyata, ini akan mengirim ke server)
    setTimeout(() => {
        const uploadedFiles = files.map(file => ({
            id: Date.now() + Math.random(),
            name: file.name,
            size: file.size,
            type: file.type,
            uploadDate: new Date().toISOString(),
            url: URL.createObjectURL(file)
        }));
        
        // Simpan ke storage
        if (!currentTeacherFiles[currentTeacherId]) {
            currentTeacherFiles[currentTeacherId] = {};
        }
        
        if (!currentTeacherFiles[currentTeacherId][fileType]) {
            currentTeacherFiles[currentTeacherId][fileType] = [];
        }
        
        // Untuk single file, replace. Untuk multiple, append
        if (validation.maxFiles === 1) {
            currentTeacherFiles[currentTeacherId][fileType] = uploadedFiles;
        } else {
            currentTeacherFiles[currentTeacherId][fileType].push(...uploadedFiles);
        }
        
        // Update tampilan
        displayUploadedFiles(fileType, currentTeacherFiles[currentTeacherId][fileType]);
        statusDiv.innerHTML = `<span class="text-green-600"><i class="fas fa-check mr-1"></i>${files.length} file berhasil diupload</span>`;
        
        // Simpan ke localStorage
        localStorage.setItem('teacherFiles', JSON.stringify(currentTeacherFiles));
    }, 1500);
}

// 4. FUNCTION untuk format ukuran file
function formatFileSize(bytes) {
    if (bytes === 0) return '0 Bytes';
    const k = 1024;
    const sizes = ['Bytes', 'KB', 'MB', 'GB'];
    const i = Math.floor(Math.log(bytes) / Math.log(k));
    return parseFloat((bytes / Math.pow(k, i)).toFixed(2)) + ' ' + sizes[i];
}

// Validasi NIK (16 digit)
function validateNIK(nik) {
    const nikRegex = /^[0-9]{16}$/;
    if (!nikRegex.test(nik)) {
        return { valid: false, message: 'NIK harus 16 digit angka' };
    }
    
    // Validasi tanggal lahir dalam NIK
    const day = parseInt(nik.substring(6, 8));
    const month = parseInt(nik.substring(8, 10));
    const year = parseInt('19' + nik.substring(10, 12));
    
    if (day < 1 || day > 31 || month < 1 || month > 12) {
        return { valid: false, message: 'Format tanggal dalam NIK tidak valid' };
    }
    
    return { valid: true, message: 'NIK valid' };
}

// Validasi NIP (18 digit)
function validateNIP(nip) {
    if (!nip) return { valid: true, message: 'NIP opsional' };
    
    const nipRegex = /^[0-9]{18}$/;
    if (!nipRegex.test(nip)) {
        return { valid: false, message: 'NIP harus 18 digit angka' };
    }
    
    return { valid: true, message: 'NIP valid' };
}

// Include crypto-js library
// <script src="https://cdnjs.cloudflare.com/ajax/libs/crypto-js/4.1.1/crypto-js.min.js"></script>

class SecurityManager {
    // Hash password menggunakan SHA-256
    static hashPassword(password, salt = null) {
        if (!salt) {
            salt = CryptoJS.lib.WordArray.random(128/8).toString();
        }
        
        const hash = CryptoJS.SHA256(password + salt).toString();
        return { hash, salt };
    }
    
    // Verifikasi password
    static verifyPassword(password, hash, salt) {
        const newHash = CryptoJS.SHA256(password + salt).toString();
        return newHash === hash;
    }
    
    // Generate secure random password
    static generateSecurePassword(length = 12) {
        const charset = 'abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789!@#$%^&*';
        let password = '';
        
        for (let i = 0; i < length; i++) {
            const randomIndex = Math.floor(Math.random() * charset.length);
            password += charset[randomIndex];
        }
        
        return password;
    }
}

// Update DataManager untuk menggunakan hashing
class DataManager {
    setUserPassword(userId, password) {
        const passwords = this.getPasswords();
        const { hash, salt } = SecurityManager.hashPassword(password);
        
        passwords[userId] = { hash, salt };
        return this.savePasswords(passwords);
    }
    
    verifyPassword(userId, password) {
        const passwords = this.getPasswords();
        const userData = passwords[userId];
        
        if (typeof userData === 'string') {
            // Backward compatibility untuk password lama
            return userData === password;
        }
        
        return SecurityManager.verifyPassword(password, userData.hash, userData.salt);
    }
}

class SessionManager {
    static SESSION_TIMEOUT = 30 * 60 * 1000; // 30 minutes
    
    static createSession(userId, userData) {
        const sessionData = {
            userId: userId,
            userData: userData,
            loginTime: Date.now(),
            lastActivity: Date.now(),
            sessionId: this.generateSessionId(),
            expiresAt: Date.now() + this.SESSION_TIMEOUT
        };
        
        localStorage.setItem('user_session', JSON.stringify(sessionData));
        this.startSessionTimer();
        
        return sessionData;
    }
    
    static validateSession() {
        const session = this.getSession();
        if (!session) return false;
        
        if (Date.now() > session.expiresAt) {
            this.destroySession();
            return false;
        }
        
        // Update last activity
        session.lastActivity = Date.now();
        session.expiresAt = Date.now() + this.SESSION_TIMEOUT;
        localStorage.setItem('user_session', JSON.stringify(session));
        
        return true;
    }
    
    static getSession() {
        try {
            return JSON.parse(localStorage.getItem('user_session'));
        } catch (e) {
            return null;
        }
    }
    
    static destroySession() {
        localStorage.removeItem('user_session');
        localStorage.removeItem('currentUser');
    }
    
    static generateSessionId() {
        return 'sess_' + Date.now() + '_' + Math.random().toString(36).substr(2, 9);
    }
    
    static startSessionTimer() {
        setInterval(() => {
            if (!this.validateSession()) {
                showNotification('Sesi berakhir. Silakan login kembali.', 'warning');
                setTimeout(() => {
                    changeUser();
                }, 2000);
            }
        }, 60000); // Check every minute
    }
}

3. Audit Trail 

class AuditManager {
    static logAction(action, details = {}) {
        const auditLog = this.getAuditLog();
        const logEntry = {
            id: Date.now(),
            timestamp: new Date().toISOString(),
            userId: currentUser,
            userName: currentUserData?.namaGuru || 'Unknown',
            action: action,
            details: details,
            ip: 'Local', // In real app, get actual IP
            userAgent: navigator.userAgent.substring(0, 100)
        };
        
        auditLog.unshift(logEntry);
        
        // Keep only last 100 entries
        if (auditLog.length > 100) {
            auditLog.splice(100);
        }
        
        localStorage.setItem('audit_log', JSON.stringify(auditLog));
    }
    
    static getAuditLog() {
        try {
            return JSON.parse(localStorage.getItem('audit_log')) || [];
        } catch (e) {
            return [];
        }
    }
    
    static exportAuditLog() {
        const auditLog = this.getAuditLog();
        const exportData = {
            auditLog: auditLog,
            exportDate: new Date().toISOString(),
            totalEntries: auditLog.length
        };
        
        downloadJSON(exportData, `audit-log-${new Date().toISOString().split('T')[0]}.json`);
    }
}

Troubleshooting

Data Tidak Tersimpan

Penyebab:

  • LocalStorage penuh atau diblokir browser
  • Mode incognito/private browsing
  • JavaScript error

Solusi:

  • Bersihkan cache browser
  • Gunakan mode normal browser
  • Periksa console untuk error
  • Pastikan JavaScript aktif

File Tidak Bisa Diupload

Penyebab:

  • File terlalu besar (>10MB)
  • Format file tidak didukung
  • Browser tidak mendukung File API

Solusi:

  • Kompres file sebelum upload
  • Gunakan format PDF untuk dokumen
  • Update browser ke versi terbaru

Tampilan Tidak Responsif

Penyebab:

  • CSS tidak termuat dengan benar
  • Viewport meta tag hilang
  • Tailwind CSS tidak aktif

Solusi:

  • Periksa koneksi internet
  • Refresh halaman (Ctrl+F5)
  • Pastikan CDN Tailwind tersedia

Lupa Password

Solusi untuk Admin:

  • Buka Developer Tools (F12)
  • Masuk ke tab Console
  • Ketik: dataManager.setUserPassword('admin', 'admin123')
  • Login dengan password: admin123

Solusi untuk Guru:

  • Hubungi administrator
  • Admin dapat reset password melalui menu Password Manager

  • Rutin melakukan backup data
  • Menggunakan password yang kuat
  • Tidak menggunakan komputer publik

  • Loading CSS dan JavaScript dari CDN
  • Font Awesome icons

  4. Semua data dan pengaturan akan dipulihkan
